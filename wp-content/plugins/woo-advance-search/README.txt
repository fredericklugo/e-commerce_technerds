Advance Search for WooCommerce plugin allows you to add an advanced search option for WooCommerce Products.

== Description ==
Advance Search for WooCommerce plugin allows you to add an advanced search option for WooCommerce Products.

With this option you can search products by product tag and category.

Yu can apply filter searcher like Title, order by date, price category and search order by ascending, Descending.

You can customize search as per your requirement like enable and disable product category and tag.

You can view searcher option by preview option. you can integrated searcher option in your site using a short-code on a page, as the widget in a sidebar or as template tag in a template.

== Installation ==

* Download the plugin
* Upload the folder "woo-advance-search" to wp-content/plugins (or upload a zip through the Wordpress admin)
* Activate and enjoy!
* You can access the setting page from here WooCommerce -> Settings -> General.
* Enable to change Woo Advance Search settings form here WooCommerce -> Settings -> General.
